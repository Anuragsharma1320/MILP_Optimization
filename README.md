# Techno-economic anaylsis of sector-coupled system
This project implements a Mixed-Integer Linear Programming (MILP) controller for optimal energy dispatch in a renewable-based microgrid. It schedules power flows between PV, Wind, BESS, heat pump, CHP and power-to-X, and the grid to minimize operational cost.
